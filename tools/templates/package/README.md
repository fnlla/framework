**FNLLA (FINELLA) PACKAGE TEMPLATE**

This is a starter template for creating new fnlla (finella) packages.

**WHAT TO UPDATE**
**-** `composer.json` name and description
**-** `src/` namespace and classes
**-** `README.md` usage examples
**-** `LICENSE` attribution if needed

**TYPICAL STRUCTURE**
```
package/
|-- src/
|-- tests/
|-- README.md
|-- LICENSE
|-- composer.json
```

**VERSIONING**
Follow SemVer and keep compatibility with `fnlla/framework ^3.0`.
