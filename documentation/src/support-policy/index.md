**SUPPORT POLICY**

This page defines the public support window for the current Finella release line.

Current supported line: 3.0.x.
Active support until 2027-04-30.
Security-only support until 2028-04-30.

Finella follows SemVer:
**-** Patch releases: bug fixes and security fixes.
**-** Minor releases: new features and deprecations without breaking changes.
**-** Major releases: breaking changes and removals.

For wider operational guidance, see `documentation/src/operations.md`.
